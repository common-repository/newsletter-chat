A lite plugin to generate today's post as newsletter for WhatsApp subscribers

== Description ==

Newsletter Chat is a lite plugin that allows you to share today's posts to your WhatsApp subscribers. Simply enter your preferred Newsletter title and Newsletter footer. The posts are automatically set up for sharing on the Newsletter Page. 

Visit the Newsletter page everyday, after posting, to share your published posts on WhatsApp. 

The newsletter page can also be shared, so that others can assist to distribute.

If posts haven't been set up, kindly update your Timezone in Settings -> General -> Timezone 

Major features in Newsletter for WhatsApp include:

* Share Posts Published Today
* Set Newsletter Title and Footer
* Link To Read Posts Below Newsletter  
* Shareable Newsletter page for fans
* Support Inclusion

 
== Installation ==
 
This section describes how to install the plugin and get it working.
 
e.g.
 
1. Upload the plugin to your Blog.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You will be redircted to the Admin menu to Set Newsletter title and footer.
4. Save changes, and visit the page to start sharing.
 
== Frequently Asked Questions ==
 
= Today's posts are not showing on Newsletter page =
 
Check your Timezone settings and update to your current location, Settings -> General -> Timezone
